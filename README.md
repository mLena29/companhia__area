# Companhia Aerea - Banco de Dados II

## Descrição 
Sistema de status de voos usando PostgreSQL + Redis + Python.

## Tecnologias
- PostgreSQL (persistencia)
- Redis (cache)
- Python (integracao)

## Como executar
1. Rode o banco.sql no PostgreSQL
2. Instale as dependencias: pip install psycopg2-binary redis
3. Execute: python3 companhia_aerea.py
